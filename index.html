Ableton

A responsive UX/UI redesign.

Let's GO

ABOUT

This was a deep dive into the site of an amazing innovative company. This was a team project. My Role involved both UX/UI.

PROBLEM

New users to the Ableton site are confused about the products and services the company provide. This is due to an inconsistent layout and unclear information and description.

APPROACH

Our team conducted five User Tests to gather the substantial data for our research. Based off of the user feedback, as a team we created a new user flow, we used card sorting and site mapping to improve the navigation, in order to design a new product page layout. 

SOLUTION

Created a responsive web layout with restructured navigation and simplified page layouts to create a stress-free experience for first-time customers.

WHAT WE DID AS A TEAM

Together we analyzed the existing site, Using Red Lining & a Heuristic Evaluation. Together we created the Proto-Persona and User Persona. As a team we made list of direct and indirect competitors for the Competitive Analysis, an Empathy Map, Card Sorting, and a Priority Matrix..,

MY ROLE

To ensure our time was used efficiently, we divided and conquered. Independently, we each conducted our own User Tests, Contributed to the Competitive Analysis, Sketches, Lo-Fi/Mid-Fi/Hi-Fi Wireframes. I personally took lead designing the entirely new Checkout Process.

GROWTH

I found it extremely useful to separately create our own visions and ideas, and then combined forces with testing and iterations. This allowed us to weed out any bugs and ultimately achieve a beautifully functional design.

KEY TAKEAWAYS

Having an open mind to compare all ideas and thoughts is the key to discovering truly beautiful design. Having a plan from the start with clear deadlines and delegation is vital for a team's success.  I thoroughly enjoyed digging into the data and doing the research to uncover our User Persona, and where we needed to improve the site the most, for their experience. I'm also intrigued by creating something better than any of the competition out there, so this was an exciting part of the process for me.
